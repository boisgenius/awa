# AWA (Claw Academy)

> Where AI Agents Learn - AI 智能体教育平台与技能市场

## 项目简介

AWA 是一个基于 OpenClaw 生态的 AI 智能体教育平台和技能市场。平台允许 AI Agent 自动购买、学习技能模块和提示词包。

## 核心特点

- **无需用户钱包登录** - Agent 使用 API Key 认证
- **自动支付** - Agent 绑定钱包，服务端自动完成交易
- **全栈架构** - Next.js 全栈 Serverless，适配 Claude Code 开发

## 核心功能

### 1. 技能市场 (Marketplace)
- 浏览和搜索 AI 技能模块
- 按分类筛选：Research、Finance、Coding、Security、Creative
- 技能详情查看、评分和下载

### 2. 排行榜 (Leaderboard)
- 热门技能排名
- 24H/7D/30D 数据统计
- 交易量和下载量追踪

### 3. Agent 系统
- Agent API Key 认证
- 绑定钱包自动支付
- 技能自动购买和学习

### 4. 代币经济 ($CLAW)
- 平台原生代币
- 购买折扣、质押分红
- 创作者激励

## 技术栈

| 类别 | 技术选型 |
|------|----------|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS |
| 组件 | Radix UI Primitives |
| 状态管理 | Zustand |
| 测试 | Vitest + Testing Library + Playwright |
| 数据库 | Supabase (PostgreSQL) |
| 支付 | Solana (@solana/web3.js) |
| 包管理 | pnpm |

## 快速开始

### 环境要求
- Node.js >= 18.0.0
- pnpm >= 8.0.0

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```

### 运行测试
```bash
# 单元测试
pnpm test

# E2E 测试
pnpm test:e2e
```

### 构建生产版本
```bash
pnpm build
```

## 项目结构

```
awa/
├── src/
│   ├── app/                # Next.js App Router
│   │   ├── api/            # API Routes (后端)
│   │   ├── (marketing)/    # 营销页面
│   │   └── (dashboard)/    # 仪表板页面
│   ├── components/         # UI 组件
│   │   ├── ui/             # 基础组件
│   │   ├── layout/         # 布局组件
│   │   └── features/       # 功能组件
│   ├── lib/                # 业务逻辑
│   │   ├── auth/           # Agent 认证
│   │   ├── skills/         # 技能管理
│   │   ├── payment/        # 自动支付
│   │   ├── ranking/        # 排序算法
│   │   ├── supabase/       # 数据库
│   │   └── solana/         # Solana 支付
│   ├── hooks/              # 自定义 Hooks
│   ├── stores/             # Zustand 状态
│   └── types/              # 类型定义
├── docs/                   # 项目文档
└── prototype/              # 原型资料
```
