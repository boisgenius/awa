# Claw Academy

> Where AI Agents Learn - AI 智能体教育平台与提示词市场

## 项目简介

Claw Academy 是一个基于 OpenClaw 生态的 AI 智能体教育平台和技能市场。平台允许用户购买、销售和管理 AI Agent 的技能模块、提示词包和教育内容。

## 核心功能

### 1. 技能市场 (Marketplace)
- 浏览和搜索 AI 技能模块
- 按分类筛选：Research、Finance、Coding、Security、Creative
- 技能详情查看、评分和下载

### 2. 排行榜 (Leaderboard)
- 热门技能排名
- 24H/7D/30D 数据统计
- 交易量和下载量追踪

### 3. 用户系统
- 钱包连接 (Solana/Base)
- 收藏和购买记录
- 创作者认证系统

### 4. 代币经济 ($CLAW)
- 平台原生代币
- 购买折扣、质押分红
- 创作者激励

## 技术栈

| 类别 | 技术选型 |
|------|----------|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS |
| 组件 | Radix UI Primitives |
| 动画 | Framer Motion |
| 状态管理 | Zustand |
| 测试 | Vitest + Testing Library + Playwright |
| 钱包 | @solana/wallet-adapter |

## 快速开始

### 环境要求
- Node.js >= 18.0.0
- pnpm >= 8.0.0

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```

### 运行测试
```bash
# 单元测试
pnpm test

# 单元测试 (watch 模式)
pnpm test:watch

# E2E 测试
pnpm test:e2e

# 测试覆盖率
pnpm test:coverage
```

### 构建生产版本
```bash
pnpm build
```

## 测试驱动开发 (TDD)

本项目严格遵循 TDD 开发流程：

1. **Red**: 先编写失败的测试
2. **Green**: 编写最小代码使测试通过
3. **Refactor**: 重构代码保持测试通过

### 测试覆盖要求
- 单元测试覆盖率 >= 80%
- 关键路径 E2E 测试覆盖
- 组件快照测试
