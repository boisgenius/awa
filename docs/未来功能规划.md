## 1. 钱包连接功能

### 设计概念
用户可通过 "Connect" 按钮连接 Solana 钱包，实现：
- 身份认证
- SOL 支付购买技能
- 查看钱包余额

### UI 设计
```
位置: Header 右侧
样式:
- 按钮: 红色背景 (#E40F3A)，白色文字
- Hover: 发光效果 + 上浮 1px
- 连接后显示: 钱包地址缩写 (0x1234...5678)
```

### 交互流程
```
点击 Connect → 弹出钱包选择 → 选择 Phantom/Solflare
    ↓
确认连接 → 按钮变为钱包地址
    ↓
点击地址 → 显示下拉菜单（余额、断开连接）
```

### 代码参考
```tsx
<button className="connect-btn">Connect</button>

// CSS
.connect-btn {
  background: var(--crimson);
  color: white;
  padding: 12px 20px;
  border-radius: 8px;
  font-weight: 600;
  transition: all 150ms ease;
}
.connect-btn:hover {
  box-shadow: 0 0 20px rgba(228, 15, 58, 0.4);
  transform: translateY(-1px);
}
```

## 2. 代币价格小部件 (Token Widget)

### 设计概念
在侧边栏底部显示平台代币实时价格信息：
- 代币符号和名称
- 当前价格
- 24小时涨跌幅
- 市值和交易量

### UI 设计
```
位置: Sidebar 底部
样式:
- 背景: 红色渐变半透明
- 边框: 红色半透明
- 价格: 大字红色
- 涨跌: 绿色(涨)/红色(跌)
```

### 数据结构
```typescript
interface TokenData {
  symbol: string;      // "$AWA" 或 "$CLAW"
  price: number;       // 0.0847
  change24h: number;   // 12.4 (百分比)
  marketCap: string;   // "$8.4M"
  volume24h: string;   // "$847K"
}
```

### 代码参考
```tsx
<div className="token-widget">
  <div className="token-header">
    <span>⚡</span>
    <span className="token-name">$AWA</span>
    <span className="token-change">+12.4%</span>
  </div>
  <div className="token-price">$0.0847</div>
  <div className="token-stats">
    <span>MCap <span className="value">$8.4M</span></span>
    <span>Vol 24h <span className="value">$847K</span></span>
  </div>
</div>
```

### CSS 样式
```css
.token-widget {
  margin: 16px;
  padding: 16px;
  background: linear-gradient(135deg, rgba(228, 15, 58, 0.15), rgba(228, 15, 58, 0.05));
  border: 1px solid rgba(228, 15, 58, 0.3);
  border-radius: 12px;
}
.token-price {
  font-size: 24px;
  font-weight: 700;
  color: #E40F3A;
}
.token-change {
  color: #00FF88; /* 正向 */
}
```

## 3. 交易量统计 (Volume Stats)

### 设计概念
在 Stats Bar 显示平台交易数据：
- 24小时交易量 (SOL)
- 地板价 (Floor Price)

### UI 设计
```
位置: Marketplace Stats Bar
样式:
- 标签: 小字灰色，大写
- 数值: 大字白色
- 单位: 小字灰色 (SOL)
- 变化: 绿色(正)/红色(负)
```

### 数据结构
```typescript
interface VolumeStats {
  volume24h: number;     // 847.2
  volumeChange: number;  // -3.2 (百分比)
  floorPrice: number;    // 0.5
  floorChange: number;   // 5.1 (百分比)
}
```

### 代码参考
```tsx
<div className="stat">
  <div className="stat-label">Volume 24H</div>
  <div className="stat-value">
    847.2 <span className="unit">SOL</span>
    <span className="change negative">-3.2%</span>
  </div>
</div>
<div className="stat">
  <div className="stat-label">Floor Price</div>
  <div className="stat-value">
    0.5 <span className="unit">SOL</span>
    <span className="change positive">+5.1%</span>
  </div>
</div>
```

## 4. 排行榜交易数据列

### 设计概念
排行榜表格显示每个技能的交易数据：
- Floor Price (地板价)
- Volume (交易量)

### 数据结构
```typescript
interface LeaderboardItem {
  rank: number;
  name: string;
  category: string;
  floor: string;      // "2.34" SOL
  volume: string;     // "1,234.56" SOL
  change24h: string;  // "+12.4%"
  change7d: string;   // "+127.4%"
  downloads: string;
}
```

### 表格列设计
```
| # | SKILL | CATEGORY | FLOOR ↕ | 24H ↕ | 7D ↕ | VOLUME ↕ | DOWNLOADS ↕ |
```

### 代码参考
```tsx
<th className="sortable">FLOOR ↕</th>
<th className="sortable">VOLUME ↕</th>

<td><span className="mono">{item.floor}</span> SOL</td>
<td><span className="mono">{item.volume}</span> SOL</td>
```

## 实现建议

### 优先级
1. **高** - 钱包连接（如需支持 SOL 支付）
2. **中** - 技能价格（如需付费功能）
3. **低** - 代币小部件（需要代币发行）
4. **低** - 交易量统计（需要交易数据）

### 技术栈建议
- 钱包连接: `@solana/wallet-adapter-react`
- 价格 API: Jupiter / Birdeye / CoinGecko
- 实时更新: WebSocket 或轮询

### 数据来源
- 代币价格: DEX API 或价格聚合器
- 交易量: 链上数据或自建统计
- 地板价: 技能列表最低价格
