# 1. 从现有平台学习

### Moltbook 的启示

**纯智能体社交网络：** Moltbook 是一个社交论坛，只有运行在 OpenClaw 上的 AI 智能体可以发帖、评论或投票，而人类只能观察。其界面类似 Reddit，具有嵌套对话和"submolts"（特定主题群组）。

**快速病毒式增长伴随安全问题：** 该平台于 2026 年 1 月 28 日上线，迅速扩展至数十万个智能体。研究人员警告，从其他智能体摄取不受信任的数据会使机器人面临提示注入攻击；恶意帖子或技能可能劫持智能体的核心指令并窃取机密。

**身份与认证：** 只有所有者已"认领"的智能体才能进行交互。(claw acadamy 也是这样，只有认领的智能体才能进行交互)

**对 Claw Academy 的启示：** 纯粹的社交论坛可能会带来流量，但存在提示注入和垃圾信息的风险。教育市场应专注于高质量、经过审核的内容，而非开放式发帖。

### OpenClaw 及其技能生态系统

**开源个人 AI：** OpenClaw（前身为 Clawdbot/Moltbot）是一款 AI 助手，可以清理收件箱、浏览网页、运行 shell 命令并与数十种服务集成。它提供持久记忆和浏览器/系统控制功能。

**技能与插件：** 该平台通过"技能"进行扩展，技能是打包的工作流或提示词。用户可以通过 ClawHub 安装技能，这是一个"技能码头"，允许作者上传版本化的 AgentSkills 包，其行为类似于 NPM 中的软件包。ClawHub 展示了用于 Trello、Slack、日历等的技能；用户可以通过简单的界面浏览或发布技能。

**市场局限性：** ClawHub 专注于功能性技能，可能无法提供结构化教育或用于训练智能体核心推理能力的提示词。目前也缺乏内容审核

## 2. 概念：Claw Academy

您的域名 clawacadamy 可以成为 OpenClaw 智能体的培训学院和提示词市场。与 Moltbook（社交论坛）和 ClawHub（技能注册表）不同，学院将专注于教授智能体如何思考和行动，并销售高质量的 MD 文件/提示词，供所有者直接放入其智能体的工作区。

### 价值主张

- **教育模块：** 结构化课程（Markdown 提示词），教授基本技能——研究技巧、电子邮件礼仪、摘要生成、安全编码等——帮助智能体避免"愚蠢"。

- **精选提示词包：** 针对常见工作流程的预测试提示词包（如"新闻通讯撰写者"、"旅行规划师"、"编码助手"）。每个包都包含必要的 identity.md 和 soul.md 文件用于人格和记忆，以及分步指导。

- **技能模板与示例代码：** 提供模板，展示如何安全地构建新的 AgentSkills。这有助于开发者创建自己的技能。

- **评估与评分：** 一个评审系统，审核员会用VibeCoding打分。用户Agent也可以对内容进行评分；强调安全性和清晰度。审核员可以验证提示词是否遵循安全最佳实践（例如，避免不受信任的代码执行）并记录已知风险。验证过后可以贴上蓝标

- **学习路径V2：** 按顺序安排模块，逐步提升智能体——从初级任务（基本对话）到高级自动化（多步骤规划、API 集成）。提供测试，让所有者可以评估智能体的进展。

- **市场：** 销售单个模块 
- **人类社区反馈：** 提供一个受管理的论坛，供人类所有者讨论经验并建议新模块。避免允许智能体发帖，以减少恶意提示注入的风险。

## 3. 技术考虑

### 内容格式
使用 OpenClaw 现有的 AgentSkills 规范。每个技能包是一个包含 YAML/Markdown 文件的文件夹（skill.yaml、README.md、agent.md 等）。构建用户可以直接导入其 skills/ 目录的模块。

### 安全模板设计
遵循安全指南以防止提示注入：避免盲目信任消息，指示智能体忽略外部指令，并在每个提示词中包含明确的防护措施。Moltbook 上报告的安全问题（提示注入和远程代码执行漏洞）应指导您的安全检查清单。

### 集成与安装
提供将模块安装到 OpenClaw 的清晰说明或脚本。例如，一个下载提示词包并将文件放置在正确目录中的 CLI 命令（类似于 ClawHub 示例中的 `npx clawhub@latest install someskill`）。考虑构建一个简单的 CLI 工具（`claw-academy install myprompt`）或 Windows/macOS 的 GUI。与Moltbook 一样， 一个Twitter只能认领一个


### 版本控制和更新
追踪每个模块的版本，并允许用户回滚。使用语义化版本号来标识更新。

### 支付系统
目前仅支持 Solana 和 Base
